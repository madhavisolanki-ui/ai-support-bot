3. HARDWARE ISSUES
For any laptop or mouse malfunctions, please submit a hardware request ticket via the 'Internal Helpdesk Portal'. Replacement hardware is typically issued within 24 hours.

4. REMOTE WORKING GUIDELINES
All employees must use the company-approved VPN (GlobalProtect) when accessing internal servers from home. Please do not store sensitive company data on personal cloud storage (Google Drive/Dropbox personal accounts).

5. SOFTWARE INSTALLATION
Employees are not permitted to install third-party software without approval from the IT Security team. Please use the 'Software Center' app installed on your machine to request new tools.
